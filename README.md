# Static Website

A modern, responsive static website built with HTML and CSS. This project features a clean design with a hero section, services showcase, team members section, contact form, and footer.

## Features

- **Responsive Design**: Fully responsive layout that adapts to different screen sizes (desktop, tablet, mobile)
- **Modern UI**
- **Navigation**
- **Hero Section**
- **Services Section**
- **Team Section**
- **Contact Form**
- **Footer**

## Technologies Used

- **HTML**: Semantic markup
- **CSS**: Modern styling with Flexbox and Grid layouts
- **Google Fonts**: Poppins font family
- **Responsive Design**: Media queries for mobile-first approach
